# 思源离开即锁  
当你离开思源笔记时锁定思源。可设置多种监听模式和自定义延迟。  
# 注意事项  
为了避免由于系统不兼容或者Electron出现bug导致循环锁屏无法解开，请先在新的工作空间尝试。确保您的系统支持该插件。   
如果还是不幸被锁，请到工作空间删除整个``siyuan_leave_to_lock``文件夹，同时，请避免在iOS和Docker等无法或难以编辑工作空间文件的操作系统或环境中使用。
